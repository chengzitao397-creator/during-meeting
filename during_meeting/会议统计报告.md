# 会议统计报告

## 一、有哪些会议（以 会议名单.csv 为准）

**会议名单** 共 **54** 个会议。

- **全国人大年度会议** — 每年3月（通常3月5日开幕）
- **全国政协年度会议** — 每年3月，与人大同期
- **中共中央全会（含三中全会）** — 每届多次，三中全会多在下半年
- **中国共产党全国代表大会** — 每5年一次，多在10-11月
- **中央经济工作会议** — 每年11-12月
- **全国金融工作会议** — 不定期，历届约隔5年（1997、2002、2007、2012、2017、2023）
- **中国进出口商品交易会（广交会）** — 每年春交会约4月中旬、秋交会约10月中旬
- **中国国际进口博览会（进博会）** — 每年11月5-10日，上海
- **三中全会** — 十一届1978-12至二十届2024-07，具体见历届时间表
- **两会** — 每年3月（人大一般3月5日开幕）
- **中央经济工作会议（各年）** — 每年12月前后，本表有2012-2025各届具体日期
- **证监会系统工作会议** — 每年1-2月
- **全国科技创新大会** — 不定期（2006、2016、2022、2023等）
- **中央城镇化工作会议** — 不定期（2013、2023等）
- **一带一路国际合作高峰论坛** — 2017、2019、2023等，日期不固定
- **陆家嘴论坛** — 每年6月
- **中国发展高层论坛** — 每年3月，多在两会后
- **中关村论坛** — 每年3-5月，北京
- **世界互联网大会（乌镇）** — 每年10-11月，乌镇
- **世界人工智能大会** — 每年7-8月，上海
- **中国国际高新技术成果交易会（高交会）** — 每年11月，深圳
- **中国国际服务贸易交易会（服贸会）** — 每年8-9月，北京
- **中国—东盟博览会** — 每年9月，南宁
- **中国国际消费品博览会（消博会）** — 每年4月，海南
- **中央政治局会议（经济形势）** — 通常每季度末（如4月、7月、10月）及年末12月分析经济、部署次年工作
- **中央政治局会议（其他议题）** — 根据需要召开
- **中国人民银行货币政策委员会例会** — 每季度一次（约季末：6月、9月、12月及次年3月前后）
- **国务院常务会议** — 一般每月2—3次，具体日期不固定
- **美联储FOMC议息会议** — 每年约8次；3月、6月、9月、12月会议附带经济展望与点阵图
- **欧洲央行货币政策会议** — 每月一次（部分会议为重大决策）
- **G20领导人峰会** — 每年一次，多在第四季度，东道国轮换
- **冬季达沃斯（世界经济论坛年会）** — 每年1月下旬，瑞士达沃斯，约一周
- **夏季达沃斯（新领军者年会）** — 每年6月或9月，中国城市（如天津、大连）
- **博鳌亚洲论坛** — 每年3月下旬，海南博鳌
- **IMF与世行春季会议** — 每年4月，华盛顿为主
- **IMF与世行秋季年会** — 每年10月，多在美国或成员国轮换
- **OPEC+部长级会议** — 每年多次，具体日期由产油国协商
- **英国央行货币政策会议** — 每月一次（部分为决议日）
- **日本央行货币政策会议** — 每年约8次
- **全国住房城乡建设工作会议** — 通常每年底或次年1月
- **中央农村工作会议** — 通常12月下旬
- **CES（国际消费电子展）** — 每年1月上旬，美国拉斯维加斯
- **日本央行1月货币政策会议** — 每年1月一次
- **全国林业和草原工作会议** — 多在每年1月
- **世界政府峰会（阿联酋）** — 每年2月，迪拜
- **巴菲特致股东信发布** — 每年2月下旬
- **MWC巴塞罗那（世界移动通信大会）** — 每年2月底，西班牙巴塞罗那
- **伯克希尔股东大会（巴菲特股东大会）** — 每年5月上旬，美国奥马哈
- **世界半导体大会（中国）** — 多在每年5月，南京等
- **中国国际大数据产业博览会（数博会）** — 每年5月，贵阳
- **G7峰会** — 每年一次，多在5-6月
- **杰克逊霍尔全球央行年会** — 每年8月下旬，美国怀俄明
- **中国国际智能产业博览会（智博会）** — 每年8月，重庆
- **世界机器人大会** — 每年8月，北京

其中，在本项目「受影响显著一级行业」中实际纳入统计的会议族群为 **16** 个（来自按月份分类表）。

## 三、是否定时召开

- **定时**：名单中「大概召开时间」为每年/每季度/每届且未标不定期、不固定。

- **不定时**：不定期、日期不固定或未明确固定周期。


- 定时：**43** 个

- 不定时：**11** 个
